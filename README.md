# Car Price Prediction System

A machine learning-based car price prediction system developed using Python and Scikit-learn. This project predicts used car prices using regression models, data preprocessing, exploratory data analysis (EDA), feature engineering, and interactive prediction functionality.

---

# Project Overview

This project uses machine learning techniques to estimate the price of used cars based on features such as:

* Car Make
* Model
* Manufacturing Year
* Kilometers Driven
* Fuel Type
* Transmission
* Engine Capacity
* Drivetrain

The project demonstrates the complete machine learning workflow including:

* Data Cleaning
* Data Preprocessing
* Exploratory Data Analysis
* Feature Engineering
* Model Training
* Model Evaluation
* Interactive Prediction

# Dataset Information

Dataset contains information about used cars including:

| Feature      | Description        |
| ------------ | ------------------ |
| Make         | Car manufacturer   |
| Model        | Car model          |
| Year         | Manufacturing year |
| Kilometer    | Distance driven    |
| Fuel Type    | Petrol/Diesel/etc  |
| Transmission | Manual/Automatic   |
| Engine       | Engine capacity    |
| Drivetrain   | Vehicle drivetrain |
| Price        | Target variable    |
